Internet

História da Internet

  • Criada durante a guerra fria (1949)
  • URSS lançou o satélite Sputnik (1969)
  • O presidente dos Estados Unidos na época (Dwight Eisenhower) criou a DARPA (Defense Advanced Research Projects Agency), agência que cria tecnologias para guerra
  • ARPANET (Advanced Research Projects Agency Network) foi a primeira rede de computadores e a base para o desenvolvimento da internet moderna

Cliente & Servidor

  • Cliente: Computador do usuário que está acessando um site por meio do navegador.
  • Servidor: O servidor é o dispositivo que hospeda o site.

Como era a Internet

  • Tim Berners-Lee criou o protocolo HTTP (Hypertext Transfer Protocol) é um padrão de mensagens que permite a troca de comunicação entre clientes e servidores da web.
  • Criou também o HTML (HyperText Markup Language)
  • Criou o WWW (world wide web)
  • Marc Andreessen criou o primeiro navegador (Mosaic) compatível com o protocolo HTTP

Como a internet funciona?

  • Através de um conjunto de protocolos e infraestrutura de rede que permite a troca de dados e mensagens entre os usuários
  • Os dados no computador são representados com 0 & 1 (Dígitos Binários ou bit)
  • A computação reúne uma porção minimo de 8 bits (byte) para a representação de algo

Representação de dados

  • 8 bits = 1 byte
  • 1024 bytes = 1 KB
  • 1024 KB = 1 MB
  • 1024 MB = 1 GB
  • 1024 GB = 1 TB
  • 1024 TB = 1 PB
  • 1024 PB = 1 EB
  • 1024 EB = 1 ZB
  • 1024 ZB = 1 YB
  • MB (MegaByte) ≠ Mb (Megabits)
  • IP: Internet Protocol
  • Servidores DNS: (Domain Name System) são máquinas que traduzem nomes de domínio em endereços IP, permitindo que os navegadores se conectem à internet

Domínio & Hospedagem

  • Domínio: É o endereço do site, que os usuários digitam na barra de endereços do navegador para acessá-lo. O domínio é um registro único que traduz o endereço IP do servidor para um nome mais fácil de memorizar.
  • Hospedagem: É o serviço que armazena os arquivos do site em servidores, tornando-os acessíveis na internet. A hospedagem é responsável por levar as informações do site até o cliente e mantê-las disponíveis 24 horas por dia.
  • TLD: Top-Level Domain, faz parte do universo de domínios da internet. A tradução literal do termo é “domínio de nível superior”, que se refere à parte do domínio que vem depois do ponto, como “. com.br” ou “. net”.

URL

  • sigla para Uniform Resource Locator (Localizador Uniforme de Recursos). Em outras palavras, é o endereço de um site ou arquivo que tem uma localização na Internet.
Ex: www.github.com/gustavoguanabara
  • url: www.github.com/gustavoguanabara
  • domínio: github.com
  • TLD: .com
  • sub-domínio: www
  • caminho: /gustavoguanabara
