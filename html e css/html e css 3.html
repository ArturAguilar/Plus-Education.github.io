Git & Github

  • Git: É um sistema de controle de versões distribuídas usado para gerenciar e gerenciar alterações no código-fonte de projetos. Ele é executado localmente no seu computador e permite:

    • Criar repositórios.
    • Rastrear alterações nos arquivos.
    • Reverter para versões anteriores.
    • Trabalhar de forma colaborativa, mesmo sem internet.


  • GitHub: É uma plataforma baseada na web que usa o Git para armazenar repositórios. Ele funciona como uma rede social para desenvolvedores, permitindo que você:

    • Armazene projetos remotamente.
    • Colabore com outros desenvolvedores.
    • Gerencie projetos usando issues, pull requests e branches.
    • Integre com outras ferramentas, como CI/CD (Integração Contínua/Entrega Contínua).
